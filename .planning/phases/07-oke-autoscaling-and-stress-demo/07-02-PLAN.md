<objective>
Ship the OCI control-plane assets needed to enable Cluster Autoscaler and
the External Metrics path for HPA RPS — without mutating live OCI state.

calls happen during an operator-gated window (per CONTEXT.md `<deferred>`);

review-then-apply step instead of a write-from-scratch step.

values) that map 1:1 to the live commands needed at apply time, all
idempotent, all dry-run by default, all scoped to the OCTO cluster.

Output: New shell script + JSON config under `deploy/oke/`; new helm
sub-chart values file under `deploy/helm/octo-apm-demo/charts/`; extended
test assertions in `tests/test_stress_demo_surface.py`.

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Local repo → OCI control plane | `configure-cluster-autoscaler.sh --apply` issues `oci ce cluster install-addon` |
| Repo file → cluster config | `cluster-autoscaler-config.json` injected via `oci ce ... --from-json` |
| HPA controller → prometheus-adapter | Adapter serves External Metrics API; HPA trusts its rule evaluation |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-05 | Tampering | configure-cluster-autoscaler.sh accidentally hardcoded `--apply` | mitigate | `: "${APPLY:=false}"` defaults to dry-run (verified by `test_configure_cluster_autoscaler_dry_run_default`); `read -p` confirm prompt blocks accidental apply |
| T-07-06 | Information Disclosure | Committed OCIDs leak compartment/pool identity | mitigate | `${OKE_NODE_POOL_OCID}` envsubst placeholder; grep gate in acceptance forbids digit patterns matching OCIDs; KB-456 / SEC-04 enforced |
| T-07-07 | Denial of Service | Misconfigured `nodes: 2:4` value over-scales cluster | accept | min=2 max=4 matches D-04 explicit; node pool quotas in OCI enforce hard cap independently |
| T-07-08 | Elevation of Privilege | Script requires OCI admin profile to install add-on | mitigate | `OCI_PROFILE:=emdemo` documented in usage; operator window gating per CONTEXT.md `<deferred>` |
| T-07-09 | Repudiation | Apply action not auditable | mitigate | `oci ce cluster install-addon` events flow to OCI audit log natively; confirm prompt forces operator to type cluster name |
</threat_model>
