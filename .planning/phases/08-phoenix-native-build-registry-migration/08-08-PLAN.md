      runbook step in 08-08).

      flip, not a manifest rewrite.
  - `## Alternatives considered`:
    - OCIR cross-region replication (rejected: not supported by OCIR Service today).
    - Local ARM builds with QEMU emulation (rejected per global CLAUDE.md "Cloud-Based Docker
      Builds" rule — slow and unreliable on Apple Silicon).
    - Status quo with frankfurt builds + cross-region pulls (rejected: fails BUILD-05 cold-pull
      target).
    - Mirror via Harbor or external pull-through cache (rejected: adds an SPOF outside OCI tenancy
      boundary, conflicts with "stay inside LogAnalytics compartment" rule from 08-CONTEXT.md).
