remaining application services: `octo-drone-shop`, `octo-apm-java-demo`, `octo-workflow-gateway`,

that authenticates via Auth Token, builds linux/amd64, and tags immutably). Frankfurt repos remain
untouched per the ≥7-day fallback window.

All registry coordinates come from env vars set by `deploy/oke/ocir-login.sh` (08-03) plus
`credentials/emdemo/outputs.json` (gitignored). The wrapper enforces `--password-stdin` semantics
inherited from 08-03 — it MUST NOT call `docker login` itself.
</objective>
